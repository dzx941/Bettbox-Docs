# 👋 欢迎使用 Bettbox

Bettbox (原名 LiClash) 是一款基于 **FlClash** 分支开发的 Android 代理工具。
### 主要特性
* **内置Mihomo内核**: 基于最新且持续更新的Mihomo(Clash Meta)内核.
* **操作简便**: 内核基于yaml配置运行. 小白用户使用机场订阅即可使用.
* **Material You 设计**: 灵感来源于 Surfboard，界面简洁美观，交互直观，为您带来现代化的视觉体验

### 🌟 为什么选择 Bettbox?
* **原生感 UI**：采用 Material You 设计，支持系统级动态取色。
* **强大内核**：内置 Mihomo (Meta) 内核，支持前沿的代理协议。
* **极致性能**：比原版更注重安卓端的后台保活与低功耗表现。
