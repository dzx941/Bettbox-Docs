Get Bettbox up and running in just four simple steps.

### 1. Install the App
Download and install the APK file. Since Bettbox is a network proxy tool, please grant the "Install unknown apps" permission if prompted during installation.

### 2. Add Configuration
* Go to **Configuration** -> Click the **+ Plus icon**.
* Select **Import from URL**, then paste your subscription link.
* Click **Save**.

### 3. Start Connection
Tap the **Start** button on the home screen. A system VPN permission request will appear; please click **OK/Allow**.

### 4. Select a Server (Node)
In the Proxy tab, you can select a specific server. By default, it's set to automatic selection. You can switch between different groups to assign specific servers to different apps.
