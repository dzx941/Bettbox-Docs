# 📘 总功能指南

Bettbox 致力于在 Android 平台上提供最优雅的代理加速体验。本指南将带你深入了解 Bettbox 的核心武器库。

---

### ⚙️ 核心代理能力
基于强大的 **Mihomo (Clash Meta)** 内核，我们支持：

| 功能模块 | 描述 |
| :--- | :--- |
| **协议全家桶** | 支持 VLESS, Trojan, Hysteria2, WireGuard, SSH 等所有主流协议。 |
| **分流规则** | 灵活的规则引擎，支持基于域名、IP、地理位置 (GEOIP) 及进程名的分流。 |

---

### 🛡️ 智能网络管理

#### 1. 应用分流
<kbd>更多</kbd> ➜ <kbd>应用分流</kbd>
> **核心逻辑**：你可以自由决定哪些 App 必须走代理（如 Twitter），哪些 App 必须直连（如 12306），从而兼顾网速与本地服务的兼容性。

#### 2. 进阶 DNS 设置
<kbd>更多</kbd> ➜ <kbd>基本配置</kbd>➜ <kbd>DNS</kbd>
> **核心逻辑**: 支持 `Fake-IP` 模式，显著提升首次连接速度

---

### 🧰 开发者与进阶工具
* **日志实时捕获**：提供全量请求日志，精准排查连接故障。
* **配置导出**：支持一键备份当前所有配置，轻松迁移到新手机。

---

> **新手建议**：初次使用时，建议保持所有设置默认，仅通过“导入订阅”开始体验。Bettbox 已预设了最优的默认参数。
