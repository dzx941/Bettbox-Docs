# 📱 应用分流 

应用分流功能允许你指定哪些 App 走代理，哪些 App 绕过代理。这对于解决“网银无法登录”、“游戏高延迟”或“本地服务访问慢”非常有用。

### 🛠 操作步骤

1. **进入设置**：在底部导航栏点击 `设置` -> `路由设置`。
2. **找到分流选项**：点击 `应用分流`。
3. **选择工作模式**：
    * **黑名单模式**：勾选的应用将**直连**（不走代理），其余走代理。
    * **白名单模式**：仅勾选的应用走代理，其余全部**直连**。
4. **搜索与勾选**：在列表中找到对应的 App（如 微信、支付宝），勾选它们。
5. **保存生效**：设置会自动保存。若 VPN 已在运行，建议点击“重启服务”以确保路由表刷新。

---

### ⚠️ 注意事项
* **系统应用**：除非你非常了解其原理，否则不建议勾选“系统组件”，这可能会导致系统更新失败或推送延迟。
