# 📥 配置订阅指南

在 Bettbox 中管理订阅的方式与 FlClash 保持高度一致。无论你是使用订阅链接还是本地文件，都能快速上手。

---

### 1. 自动同步（通过 URL 导入）

这是最推荐的使用方式，支持自动更新节点。

**操作步骤：**

1.  **复制链接**：从你的服务提供商处获取 Clash 订阅链接。
2.  **打开配置**：进入 Bettbox 底部导航栏的 <kbd>配置</kbd> 页面。
3.  **点击添加**：点击右下角的 <kbd>+</kbd> 按钮，选择 <kbd>从 URL 导入</kbd>。
4.  **保存配置**：输入名称及 URL，设置更新间隔。
5.  **激活**：在列表中点击该配置卡片，确认其处于选中状态。

---

### 2. 手动导入（通过文件导入）

如果你已经下载了 `.yaml` 格式的配置文件，可以使用此方法。

**操作步骤：**

1.  点击 <kbd>+</kbd> 按钮，选择 <kbd>从文件导入</kbd>。
2.  在弹出的系统文件选择器中，定位并选中你的配置文件。
3.  导入后，该文件将存储在 Bettbox 的本地目录中。

---

### 3. 扫描二维码导入 (QR Code)

**操作步骤：**
1. 进入 Bettbox 底部导航栏的 <kbd>配置</kbd> 页面。
2. 点击右下角的 <kbd>+</kbd> 按钮，选择 <kbd>扫描二维码</kbd>。
3. **两种识别方式**：
    * **直接扫码**：将摄像头对准电脑屏幕或其他设备上的订阅二维码。
    * **相册识别**：如果二维码已保存在手机里，点击扫描界面的 <kbd>图片图标</kbd> 即可从相册导入。
4. 识别后确认名称，点击 <kbd>保存</kbd>。

---

### 4. 进阶：编辑配置文件

Bettbox 内置了轻量化的 YAML 编辑器，允许你直接微调配置。

* **如何操作**：在配置列表中，点击卡片右上角的 **更多 (三个点)** -> <kbd>编辑文件</kbd>。
* **适用场景**：手动添加 DNS 规则、修改分流模式或调整自定义策略组。

---

### ⚠️ 常见问题排查

| 现象 | 可能原因 | 解决方法 |
| :--- | :--- | :--- |
| **导入时提示解析失败** | 链接格式非标准 Clash 格式 | 尝试使用订阅转换器处理后再导入。 |
| **更新订阅超时** | 当前网络无法连接至订阅服务器 | 检查网络连接，或尝试更换 DNS 后再试。 |
| **节点显示为 0** | 配置文件内无有效 Proxy 字段 | 确认订阅是否已过期，或联系服务商。 |

---

### 💡 与 FlClash 的兼容性说明

由于 Bettbox 基于 FlClash 开发，两者在配置层面上：
* **100% 兼容**：所有在 FlClash 上能运行的配置，在 Bettbox 上均可直接使用。
* **逻辑相同**：导入、测速、切换节点的逻辑完全一致，无任何迁移成本。
* **更强性能**：Bettbox 针对大体积配置文件的加载速度进行了底层优化。
